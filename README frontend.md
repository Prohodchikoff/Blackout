# blackout frontend React app

## How to run
1. Clone repository
```
git clone https://github.com/WebUraXalys/blackout.git
```
2. Instal node and npm
```
sudo apt install nodejs
```
```
sudo apt install npm
```

2. Switch directory
```
cd frontend
```

3. Install all packages
```
npm i
```

4. Runing server

```
npm run dev
```
